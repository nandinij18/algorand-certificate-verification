# Algorand Certificate Verification System

## Description
This project is a blockchain-based certificate verification system built using Algorand.
It allows institutions to issue digital certificates that can be verified securely.

## Objective
To prevent fake certificates using blockchain technology.

## Technologies Used
- Algorand
- Python SDK
- AlgoKit

## Features
- Issue certificates
- Verify certificates
- Tamper-proof system

## Setup Instructions
1. Install AlgoKit
2. Clone the repository
3. Run the project
